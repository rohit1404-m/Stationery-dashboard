#  Stationery Sales Insights Dashboard

##  Overview

This project is an interactive Power BI dashboard designed to analyze and visualize stationery sales data. It helps in understanding business performance through key metrics such as total sales, profit, orders, and category-wise distribution.

The dashboard provides a clear view of trends across different cities, product categories, and time periods, enabling better decision-making.

##  Features

* 📍 City-wise filtering for focused analysis
* 📈 Monthly sales trend visualization
* 🧾 Category-wise sales and profit breakdown
* 📊 KPI cards for quick insights:

  * Total Sales
  * Total Profit
  * Total Orders
  * Profit Percentage
  * Average Order Value
* 🎯 Clean and minimal user interface

## 📂 Dataset

The dataset used in this project includes:

* Sales transactions data
* Product categories (Accessories, Notebooks, Markers, Pens, Files)
* City-wise sales information
* Order details


---

## 📊 Key Insights

* Accessories and Notebooks contribute the highest sales
* Profit varies across different cities
* Certain months show noticeable spikes in sales trends
* Some categories generate high sales but lower profit margins

## ▶️ How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Use filters (like city selection) to explore the dashboard
4. Interact with visuals to gain insights
